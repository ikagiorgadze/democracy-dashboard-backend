Retrieve ranked dataset correlations for a country

query Parameters
country
required
string

Human-readable country label matching parquet directory naming.

type
required
string (CorrelationRankingType)
Enum: "highest" "lowest" "strongest" "weakest" "most_significant" "least_significant" "most_observations" "fewest_observations"
dataset1
required
string (DatasetCode)
Enum: "VDEM" "WEO" "NEA"
dataset2
required
string (DatasetCode)
Enum: "VDEM" "WEO" "NEA"
minObservations
integer >= 1

Minimum observation count (n) to include in results.

limit
integer >= 1

Optional override for result count (defaults internally).
